# MEM Baby Lab

This project explores how systems learn when they do not fully understand what they are observing.

Instead of relying on predefined categories, the system operates only on signal behavior over time.

## Core Idea

Most systems assume they know what they are looking at.

This environment removes that assumption.

The system receives:
- no labels
- no predefined structure
- no semantic meaning

Only signal dynamics.

## What this repository contains

- Simulation environment
- Adaptive learning behavior
- Evolution scenarios
- Output analysis

## Purpose

This is part of ongoing research into:

- learning under uncertainty
- decision behavior without predefined categories
- adaptive systems

## Note

This repository is a research environment.

It does not represent a full production system.

Underlying decision principles and architecture are not included.
